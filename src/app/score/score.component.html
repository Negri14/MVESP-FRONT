O Score de Violência


O Score de Violência foi criado com o objetivo de facilitar a identificação de regiões mais violentas, utilizando o código penal para fazer a definição de crimes mais graves ou menos graves. Utilizamos como critério para definir o impacto de cada indicador no score de violência a pena média prevista no Código Penal. Dessa forma, foi possível calcular o score multiplicando a incidência do crime a cada 100 mil habitantes pelo coeficiente normalizado para cada tipo de crime.
